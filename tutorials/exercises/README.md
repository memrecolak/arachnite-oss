# Exercises

These small challenges let you practice what you learned in the lessons.

Each exercise has two files:
- `exercise_NN_*.py` — a starter file with `TODO` comments. Fill in the
  blanks.
- `solutions/exercise_NN_*_solution.py` — a complete worked answer. Only
  peek if you're truly stuck.

## How to use them

1. Open the starter file.
2. Read all the comments first.
3. Fill in the `TODO` parts.
4. Run the file.
5. If it doesn't work the way you expect, read the error and try to fix it.
6. If you get stuck for more than 15 minutes, look at the solution.
7. After looking, **close the solution and try again from scratch.**

## The exercises

| # | File | What you'll practice |
|---|---|---|
| 1 | [exercise_01_hello_agent.py](exercise_01_hello_agent.py) | Build the simplest agent: one sensor, one instinct, one action |
| 2 | [exercise_02_two_sensors.py](exercise_02_two_sensors.py) | Combine readings from two sensors in one instinct |
| 3 | [exercise_03_priority_battle.py](exercise_03_priority_battle.py) | Watch two instincts compete; pick the right priorities |
| 4 | [exercise_04_safety_reflex.py](exercise_04_safety_reflex.py) | Add a reflex that overrides normal instincts |

Solutions live in [`solutions/`](solutions/).

Have fun, and remember: **you learn programming by writing programs**, not
by reading them. Type the code yourself, even when it feels slow. It's the
best way to make it stick.
